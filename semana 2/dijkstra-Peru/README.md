# Algoritmo de Dijkstra - Mapa del Perú

Este proyecto implementa un sistema de búsqueda de rutas mínimas entre las capitales de los 25 departamentos del Perú.

## Especificaciones Técnicas
- Algoritmo: Dijkstra 
- Cálculo de Distancia: Fórmula de Haversine (distancia del gran círculo entre coordenadas geográficas).
- Interfaz: Google Maps API para la visualización dinámica.
- Datos: Se han mapeado 25 nodos (capitales) y sus conexiones viales principales (aristas).
- Lenguaje de programacion: JavaScript
- Maquetado: HTML
- Estilos: CSS

## Requisitos Cumplidos
- [x] Implementación de las 25 regiones del Perú.
- [x] Uso de coordenadas reales (Latitud y Longitud).
- [x] Panel lateral con el desglose del recorrido y distancia total en KM.
